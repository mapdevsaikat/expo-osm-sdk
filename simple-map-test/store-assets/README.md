# Play Store graphics — Expo OSM Demo

Upload these in **Play Console → Grow users → Store presence → Main store listing**.

| Asset | File | Required? |
|-------|------|-----------|
| App icon (512×512) | `app-icon-512.png` | **Yes** |
| Feature graphic (1024×500) | `feature-graphic-1024x500.png` | **Yes** |
| Phone screenshots (≥2) | `phone-screenshots/01-*.png` … `03-*.png` | **Yes** |
| 7-inch tablet | — | **Optional** (phone-only app) |
| 10-inch tablet | — | **Optional** (phone-only app) |

## Tablet screenshots

Not required if you limit the app to **phones only**:

**Device catalog → Manage devices → exclude tablets** (or leave tablet slots empty — Play allows this for phone-focused apps).
